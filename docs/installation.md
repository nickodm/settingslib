# Installation of Settingslib

To install the library, use the `git clone` option.

# Step by Step

1. Go to the [project's GitHub](https://github.com/nickodm/settingslib), click on the `Code` green button and then copy the SSH link.
![Menu of `Code` green button](./assets/installation/GithubCodeSSH.png)

2. Then, open your terminal and write the next command:
```bash
git clone git@github.com:nickodm/settingslib.git
```
As soon as you press `Enter`, [Git](https://git-scm.com/) will ask you for your passphrase.
```
cloning into 'settingslib'...
Enter passphrase for key:
```
